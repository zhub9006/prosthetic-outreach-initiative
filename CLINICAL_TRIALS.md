## 🔴 Tier 1: Directly Affordable Prosthetic Solutions

### 1. Low Cost Socket for Lower Limb Amputees
| Field | Details |
|---|---|
| **NCT ID** | [NCT04725461](https://clinicaltrials.gov/study/NCT04725461) |
| **Title** | Low Cost Prosthetic Socket for Lower Limb Amputees |
| **Sponsor** | Shirley Ryan AbilityLab (Chicago, IL!) |
| **PI** | Laura Miller |
| **Status** | ✅ COMPLETED (Jul 2018 – Jan 2025) |
| **Enrollment** | 8 |
| **Phase** | N/A (Device feasibility) |

**Key Innovation**: Simplified socket fitting protocol using expandable rigid foam. Cast limb in <30 min, finish socket in 90 min — enabling one-visit fitting for low-resource settings. Uses 3D-printed outer shell + expanding polyurethane foam (Foam IT 8). Compatible with ICRC transtibial polypropylene component system.

**Why It Matters**: Directly addresses the need for high-quality, affordable prosthetic technology in developing countries. Patent-pending method. Perfect model for outreach: fast, low-cost, no multi-visit requirement.

### 2. 3D-Printed Prosthetic Sockets for Affordable Care
| Field | Details |
|---|---|
| **NCT ID** | [NCT03517774](https://clinicaltrials.gov/study/NCT03517774) |
| **Title** | Improving Prosthetic Care for Patients With a Lower Limb Amputation |
| **Sponsor** | Sunnybrook Health Sciences Centre |
| **Technology Partner** | Nia Technologies (3DPrintAbility) |
| **Status** | ✅ COMPLETED (May 2018 – Mar 2021) |
| **Enrollment** | 12 |
| **Phase** | N/A |

**Key Findings**:
- 3D-printed prosthetic sockets performed at the **same efficacy** as traditionally manufactured prosthetics
- Fitting time cut from ~5 days to **under 1.5 days**
- Prior validation in Cambodia, Tanzania, Uganda (2016–2017) with pediatric transtibial amputees
- 3D printed sockets equivalent in safety, fit, durability, and comfort to standard thermoplastic sockets
- Eliminates plaster casting → reduces material & labor costs

### 3. 3D Scanning + AI Custom Socket (Upper Limb)
| Field | Details |
|---|---|
| **NCT ID** | [NCT07437664](https://clinicaltrials.gov/study/NCT07437664) |
| **Title** | Computer-Aided Design for Customized Soft Transradial Prosthetic Sockets Using 3D Scanning |
| **Sponsor** | Al-Nahrain University (Baghdad, Iraq) |
| **PI** | Amjed Abdulameer Hussein |
| **Status** | 🟢 ACTIVE NOT RECRUITING (Start: Feb 2026) |
| **Enrollment** | 1 |
| **Keywords** | Robotic Hand, AI, 3D Scanning, 3D Printing, **Low-cost Prosthetics**, EMG |

**Key Innovation**: Low-cost, AI-powered prosthetic system with 3D-scanned custom sockets. AI classifies EMG signals for adaptive grasping. Compares AI-driven control vs. traditional rule-based programming.

**Why It Matters**: Demonstrates how AI + 3D printing can dramatically reduce prosthetic costs while improving functionality. Open methodology could be replicated by affordable makerspaces.

### 4. ORTHOPUS — Open-Source Affordable Upper Limb Prosthetics for Nepal
| Field | Details |
|---|---|
| **NCT ID** | [NCT04803253](https://clinicaltrials.gov/study/NCT04803253) |
| **Title** | Social and Professional Reintegration Using Affordable Solutions for Upper Limb Amputation |
| **Sponsor** | Orthopus (French company, social impact model) |
| **Collaborator** | Handicap International (Nepal) |
| **Status** | ❓ UNKNOWN (was RECRUITING) |
| **Enrollment** | 14 |
| **Location** | Nepal |
| **Ethics Approval** | Yes (DMC) |

**Key Innovations**:
- Set: aesthetic articulated hand + mechanical wrist + heavy-duty work hook + body-powered harness
- **Open-source / frugal innovation model** — all solutions shared via free/open-source license
- CE marking (European quality standards) at moderate pricing
- Designed specifically for LMIC constraints (supply + cost)
- Targets: 1% of Nepalese population has disability; 73% lack assistive devices

**Why It Matters**: Orthopus directly models how to provide affordable, quality prosthetics in resource-limited settings. Their open-source approach is perfect for scaling outreach.

## 🟡 Tier 2: Related Prosthetic & Rehabilitation Technologies

### 5. SMART mHealth — Self-Management for Amputee Rehabilitation (UBC)
| Field | Details |
|---|---|
| **NCT ID** | [NCT04953364](https://clinicaltrials.gov/study/NCT04953364) |
| **Title** | Self-Management for Amputee Rehabilitation Using Technology (SMART) |
| **Sponsor** | University of British Columbia |
| **Collaborator** | CIHR |
| **Status** | 🟢 RECRUITING |
| **Enrollment** | 82 (estimated) |
| **Start Date** | Oct 2022 |
| **Study Type** | RCT (6-week intervention + 4-week retention) |

**Key Innovation**: Tablet-based mHealth platform for amputee self-management — education, prosthetic limb management, peer support. Targets rural/underserved populations with limited healthcare access. Combines virtual coaching with peer support.

**Why It Matters**: Tech-assisted approach can extend rehabilitation reach to rural and low-income populations — directly relevant to outreach goal of expanding access.
