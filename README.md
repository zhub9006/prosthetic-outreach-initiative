# Prosthetic Outreach Initiative

A centralized project for coordinating volunteer teams across multiple regions to improve affordable prosthetic access through clinical research, community partnerships, and direct outreach.

## Regions
- **Chicago** (Midwest)
- **Boston** (Northeast)
- **Atlanta** (Southeast)

## Quick Start
1. Check issues labeled `high-priority` for urgent tasks
2. Join a regional team under the appropriate `regional` + city labels
3. Track clinical trial findings and recommend low-cost prosthetic solutions
4. Coordinate meeting points for cross-regional visits and joint events

## Volunteer Task Categories
- Clinical trials identification & analysis
- Community partnerships & provider outreach
- Regional outreach event planning
- Volunteer scheduling & coordination

## GitHub Project Board
Use issues and labels to organize all cross-team work.
